# 📍 Tammeny

Secure real-time location sharing platform built with modern web technologies and a clean user experience.

---

## ✨ Overview

Tammeny is a modern location-sharing platform designed to help friends and families stay connected safely and efficiently.

The project focuses on privacy, simplicity, responsiveness, and real-time user experience.

---

## 🚀 Features

- 📍 Live location sharing
- 🔐 Secure user experience
- 📱 Responsive UI
- 🌙 Modern design system
- ⚡ Fast performance
- 👥 Friend & family connectivity
- 🗺 Interactive map experience
- 🌍 Arabic-friendly interface

---

## 🛠 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
- Modern UI Components
